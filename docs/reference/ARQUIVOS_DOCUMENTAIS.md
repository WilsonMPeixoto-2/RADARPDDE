# Arquivos documentais do RADAR PDDE

Este diretório reúne os artefatos canônicos do projeto:

- `RADAR_PDDE_Dossie_Contexto_Regras_Decisoes_v1_0.docx` — contexto, regras de negócio e decisões consolidadas;
- `RADAR_PDDE_Plano_Lote_2_APROVADO_v2_0.docx` — plano funcional, visual e de navegação aprovado;
- `RADAR_PDDE_Prototipo_Exportacao_Conservadora_v2-1.xlsx` — referência congelada para a exportação em Excel;
- `../reports/RADAR_PDDE_Relatorio_Guia_Ciclo_A_v1_0.docx` — relatório funcional e guia de uso do Ciclo A.

A precedência entre documentos está definida em `STATUS_DOCUMENTOS.md` e no README principal.
